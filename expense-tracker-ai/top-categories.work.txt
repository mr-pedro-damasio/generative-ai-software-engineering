Feature: Top Expense Categories
Branch: feature/top-categories
Worktree: /workspaces/expense-tracker-top-categories

## What Was Implemented
A "Top Categories" analytics tab added to the ExpenseTracker app. It shows all expense
categories ranked by total spending, with visual progress bars, percentages, and counts.

## Files Created
- components/TopCategories.tsx (new)
  - Accepts `expenses: Expense[]`
  - Ranks categories by total spending (highest first), skips categories with $0
  - Displays: rank (#1, #2...), category icon, name, expense count, % of total, total amount
  - Progress bar sized relative to the top category (100% width = max spend)
  - Empty state when no expenses exist
  - Uses existing CATEGORY_ICONS, CATEGORY_COLORS, getTotalByCategory from lib/utils.ts

## Files Modified
- app/page.tsx
  - Added `import TopCategories from '@/components/TopCategories'`
  - Extended tab state type: `'dashboard' | 'expenses' | 'categories'`
  - Added "🏷️ Top Categories" button to tab bar
  - Added `tab === 'categories'` branch rendering `<TopCategories expenses={expenses} />`

## Dependencies Added
None. Reuses existing utilities and Tailwind CSS.
